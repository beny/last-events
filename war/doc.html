Zadání

Cílem projektu je implementovat aplikaci, která bude zobrazovat na mapě hudební koncerty zanesené na službě Last.fm a dostupné pomocí jejich API. Aplikace bude umožňovat:

  • možnost vyhledávání podle geografické lokace
  • filtrování na základě kritérií u vystupujících umělců
  • vizualizace koncertních šňůr, zobrazení chronologie a směru

Implementace

Projekt byl implementován v App Engine v jazyce Java. Jedná se o tzv. mashup. Mashup je webová aplikace, která v rámci poskytování nových služeb používá a kombinuje data, prezentace, nebo funkcionalitu z dvou anebo více zdrojů. Samotná aplikace se skládá ze dvou částí: backend a frontend. Backendová část je celá implementována v jazyce Java. Zpracovává dotazy zaslané z frontendové části. Stahuje data z Last.fm API a cachuje je v lokálním uložišti (Storage). Pro komunikaci mezi frontend a backend částmi jsme použili technologii REST. Frontendová část zašle dotaz v podobě URL adresy a backendová část vrátí soubor ve formátu JSON, ve kterém je seznam odpovídajících koncertů. Backendová část při dotazu nejdříve zkontroluje cache. Pokud je požadovaný záznam uložen v cache a není zastaralý, vrátí data z cache. V opačném případě použije Last.fm API a načte požadováná data z něj. Last.fm API vrací XML soubor se seznamem koncertů. XML soubor je zpracován parserem, získaná data se uloží do cache a backend vrátí odpovídající JSON soubor se seznamem koncertů. Backendová část je implementována v jazyce JavaScript. Využívá Goodle Maps API. Zobrazuje informace v podobě HTML stránky.

Dotazy pro REST:
/api/artist/{query}
/api/artist/{query}/{count}
/api/location/{query}
/api/location/{query}/{distance}
/api/location/{query}/{distance}/{count}
/api/search/artist/{query}
/api/search/artist/{query}/{count}
/api/search/location/{query}
/api/search/location/{query}/{count}

Návod k použití aplikace

Aplikace umožnuje dva typy vyhledávání koncertů: podle místa a podle umělce. Uživatel vybere požadovaný typ vyhledávání, zadá klíčové slovo a tlačítkem Enter odešle dotaz. Uživatel si dále může vybrat počet zobrazených koncertů (pomocí posuvníku) a v případě vyhledávání podle lokace také vzdálenost od místa v kilometrech (pomocí posuvníku). Po potvrzení aplikace zobrazí nalezené koncerty na mapě a vypíše jejich seznam. Po najetí mysí na název koncertu se zobrazí dialogové okno s detailními informacemi.
